MAIN COLOR ANALYZER:
This was developed for the company Tint. 
This function determines the main color (not average) of images parsed from Amazon.com
It is designed to ignore white backgrounds. 

--
WEB PARSER:
This is a web parser developed for the company Tint. 
This parser returns several parameters for the top 50 clothing items from a user specified search.

Data was returned as a json object.

Developed as an alternative to the AWS Product Advertising API for server-side clothing search.
